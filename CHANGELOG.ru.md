# Журнал изменений

## [2.3.4] — текущая версия

### Совместимость с ядром Linux
- Добавлена поддержка Linux 6.16 (обновление META и исправления совместимости)
- Исправлена сборка вне исходного дерева (`out-of-src builds`)
- Устранена проблема совместимости с toolchain SIMD (XSAVE)

### Исправления файловой системы
- `zfs_putpage`: исправлен режим синхронной/асинхронной записи страниц
- `zfs_putpage`: корректное завершение асинхронного writeback
- `zvol`: устранено падение при использовании blk-mq на Linux 4.19
- `zdb`: исправлена работа с флагом `-k` для пулов

### Управление памятью
- Разрешено использование vmem_alloc-backed multilists
- Подавлено ложное предупреждение о крупных выделениях памяти ZSTD

### Сборка и пакетирование
- RPM: устранено дублирование `/sbin/zgenhostid` в `%files`
- CI: удалены устаревшие Debian backports

### Тестирование
- Добавлен тест `mmap_ftruncate` для проверки асинхронного writeback
- Улучшено логирование тестов: добавлены микросекундные временны́е метки

### FreeBSD
- Исправлена логика `zfs_putpages`: страницы не снимаются с dirty-состояния до завершения записи

### Мониторинг и уведомления
- `zed`: улучшено форматирование Slack-уведомлений

---

## [2.3.3] — базовая версия форка

Форк создан на основе ветки `zfs-2.3.4-staging` проекта OpenZFS.

Включает все исправления и улучшения OpenZFS 2.3.x:
- Полная поддержка Linux 4.18 — 6.15
- Нативное шифрование AES-256-GCM/CCM
- ZSTD-компрессия
- Поддержка BLAKE3 в качестве алгоритма контрольных сумм
- Улучшенная поддержка FreeBSD 14/15
- Многочисленные исправления стабильности и производительности
